# Python Caro Game

A Gomoku (Caro) game developed in Python using Minimax and Heuristic Evaluation.

## Features

- PvP Mode (6x6)
- PvP Mode (8x8)
- AI Mode (9x9)
- Minimax Algorithm
- Heuristic Evaluation Function
- Tkinter GUI

## Files

- AI_Caro.py : AI opponent using Minimax + Heuristic
- Caro_6x6.py : PvP version
- Caro_8x8.py : PvP version

## How to Run

```bash
python AI_Caro.py
```
